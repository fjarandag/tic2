Esta página corresponde al ejercicio de TIC-II de realizar un sitio web sencillo.

  • Basándonos en una plantilla preexistente. Se trata de un layout basado en el atributo float. La he adaptado un poco según mi idiosincrasia.
  • Incluir algunos de los estilos básicos de formato em/i , strong/b, etc
  • Incluir algún enlace (utilizar atributo title si aporta algún significado)
  • Incluir alguna lista y/o alguna tabla para mostrar información estructurada
  • La plantilla incluye una página para imágenes y otra página para un formularios
    • Las imágenes deberían incluir alguna imagen con fondo transparente (.gif y .png lo permiten)
    • El formulario aquí utilizará method="GET" action="formulario.htm" de modo que podremos ver las entradas en la URL.
  • Considerar que lo importante es mostrar que somos capaces de aplicar elementos de estilo/semántica con criterio. Esto se consigue mejor con un texto breve, contextualizado, que incluya un par de los elementos pedidos, que con un mamotreto con cada uno de los elementos del HTML 5.1 que podríamos haber copiado de cualquier lado.
